## Description
映画の名前を入れるとポスターを見つけてくれるサイトです.  

## Requirement
- create-react-app
- typescript

## Usage
1. clone
```bash
$ git clone https://github.com/FukeKazki/Movie-Search-TS.git
```
2. run
```bash
$ yarn install
$ yarn start
```
